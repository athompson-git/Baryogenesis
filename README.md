# Baryogenesis


We use a model of two color-triplet, iso-singlet scalars $X_\alpha$ with
hypercharge assignment $Y = + 4/3$ (in the $Q = Y/2$ convention)
and a Majorana fermion $\psi$.

### References

* [1304.0711](https://arxiv.org/abs/1304.0711)
* [1712.02713](https://arxiv.org/abs/1712.02713)
* [2409.08178](https://arxiv.org/abs/2409.08178)

It corresponds roughly to the scalar leptoquark $\bar{S}_1$ identified in
[1603.04993](https://arxiv.org/abs/1603.04993).
